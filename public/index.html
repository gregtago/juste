        Mode d'emploi

        Déposez une photo ou un scan d'une pièce d'identité (carte nationale, passeport) ou d'un extrait de comparution. 
        Vous pouvez également faire une capture d'écran et la déposer directement. 
        L'outil génère autant de fiches que de personnes identifiées dans le document. 
        Téléchargez le fichier XML puis importez-le dans iNot via Fichier → Import généalogiste.
      


      

        Traitement des données

        Les documents déposés sont transmis à l'API Anthropic pour analyse et ne sont jamais écrits sur disque. 
        Anthropic ne conserve pas les données au-delà de 30 jours et ne les utilise pas pour entraîner ses modèles. 
        L'application est hébergée sur Render (États-Unis). 
        Les transferts sont encadrés par les clauses contractuelles types de la Commission européenne.
